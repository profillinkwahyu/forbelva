# Digital Love Letter

A beautiful, interactive digital love letter website that combines romantic design with interactive elements to create a memorable digital gift for someone special.

## Features

1. **Landing Page**: Interactive greeting with persuasive button functionality
2. **Captured Memories**: Photo gallery with polaroid-style frames and photobooth strip
3. **Virtual Flowers**: Animated digital bouquet with flower meanings
4. **Music Player**: Custom player for "Ada Untukmu" by Budi Doremi with secret message
5. **Love Letter**: Beautifully formatted digital letter with print functionality

## Technologies Used

- HTML5, CSS3, JavaScript
- Bootstrap 5 for responsive layout
- Font Awesome for icons
- Google Fonts (Dancing Script, Poppins)
- Custom CSS animations and effects

## How to Use

1. Open `index.html` in a web browser
2. Click "Yes Please" on the landing page (or have fun with the "No Thanks" button)
3. Navigate through different sections using the top menu
4. Interact with photos, flowers, and music player
5. Print the love letter if desired

## Customization

1. Replace placeholder images in the photo gallery with your own memories
2. Update the love letter text with your personal message
3. Change colors in the CSS variables at the top of `styles/main.css`
4. Update the YouTube embed link to your special song
5. Add more flowers or change their meanings
